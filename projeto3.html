Projeto 3: ToDo List

Um painel interativo para controle de tarefas, desenvolvido com React.

Descrição do Projeto

O ToDo List é uma aplicação web que permite aos usuários criar, gerenciar e acompanhar suas tarefas diárias de forma interativa e visualmente agradável.

Tecnologias Utilizadas

  • React

Recursos e Funcionalidades

  • Criação e exclusão de tarefas
  • Organização das tarefas por categorias ou prioridades
  • Marcação de tarefas concluídas
