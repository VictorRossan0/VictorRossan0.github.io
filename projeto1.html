Projeto 1: Sistema de Gestão de Atestados

Uma aplicação completa de gerenciamento de atestados utilizando Laravel e MySQL.

Descrição do Projeto

O Sistema de Gestão de Atestados é uma ferramenta desenvolvida para cadastro e análise de atestados médicos por parte dos colaboradores e supervisores. Com essa aplicação, os colaboradores podem cadastrar seus atestados, enquanto os supervisores têm acesso a uma área para análise e aprovação/reprovação dos atestados.

Tecnologias Utilizadas

  • Laravel
  • MySQL

Recursos e Funcionalidades

  • Cadastro de atestados pelos colaboradores
  • Análise e aprovação/reprovação de atestados pelos supervisores
  • Gerenciamento e armazenamento seguro dos atestados
