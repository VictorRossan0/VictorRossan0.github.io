Projeto 2: Ferramenta de Web Scraping

Automação de coleta de dados da web usando Python e Selenium.

Descrição do Projeto

Esta ferramenta tem como objetivo consultar em cada Ficha Técnica dos CIRs os respectivos dados necessários para prosseguir com a abertura de WF.

Tecnologias Utilizadas

  • Python
  • Selenium

Recursos e Funcionalidades

  • Coleta automatizada de dados da web
  • Extração dos dados necessários das Fichas Técnicas dos CIRs
  • Processamento e manipulação dos dados coletados
  • Integração com a abertura de WF
